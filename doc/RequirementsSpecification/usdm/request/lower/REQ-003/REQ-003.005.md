request_id: REQ-003.005
level: lower
title: マッパーの解析
## demand:
  ①何が（何を）: DTO↔Entity変換層の依存関係（MapStruct等）を
  ②どんな状態／条件で: `.java`ソースコードの@Mapperを解析する際に
  ③どうする: 抽出・記録する
  文: DTO↔Entity変換層の依存関係（MapStruct等）を、`.java`ソースコードの@Mapperを解析する際に抽出・記録する
## reason:
  - マッパーの依存関係を可視化することで、DTOとEntityの変換関係を把握できる
## description:
  - 用語: マッパーとは、DTO↔Entity変換層の依存（MapStruct等）を指す
  - 範囲: JavaParserを使用して`@Mapper`/`@Mapping`とメソッドシグネチャで変換関係を記録する
  - 例: `@Mapper public interface UserMapper`の場合、UserMapper→DTO/Entityの依存関係が存在する


## specification_note: 
  - 解析タイプ:  静的解析         
  - 検索対象:  `.java`（@Mapper）                       
  - 解析ツール:  JavaParser                      
  - 解析内容:  `@Mapper`/`@Mapping`とメソッドシグネチャで変換関係を記録
