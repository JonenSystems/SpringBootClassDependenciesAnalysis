## demand:
  ①何が（何を）: `Authorization: Bearer`ヘッダ処理への依存関係を
  ②どんな状態／条件で: `.java`ソースコードのFilterまたはHTTPトレースを解析する際に
  ③どうする: 抽出・記録する
  文: `Authorization: Bearer`ヘッダ処理への依存関係を、`.java`ソースコードのFilterまたはHTTPトレースを解析する際に抽出・記録する
## reason:
  - トークン抽出の依存関係を可視化することで、JWT認証の実装を把握できる
## description:
  - 用語: トークン抽出とは、`Authorization: Bearer`ヘッダ処理への依存を指す
  - 範囲: JavaParser（静的）でヘッダ読取/前置詞`Bearer`検出。必要ならSpring Cloud Sleuth（動的）で実行時トレースで確認する
  - 例: Filterで`Authorization`ヘッダから`Bearer`トークンを抽出する場合、トークン抽出処理が記録される


## specification_note: 
  - 解析タイプ:  静的解析 or 動的解析 
  - 検索対象:  `.java`（Filter）/HTTPトレース               
  - 解析ツール:  JavaParser, Spring Cloud Sleuth 
  - 解析内容:  ヘッダ読取/前置詞`Bearer`検出。必要なら実行時トレースで確認
