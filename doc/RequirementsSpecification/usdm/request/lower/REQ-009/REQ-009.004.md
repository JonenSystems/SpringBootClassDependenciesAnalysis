request_id: REQ-009.004
level: lower
title: パス/パラメータ依存の解析
## demand:
  ①何が（何を）: `@PathVariable`/`@RequestParam`などの契約依存関係を
  ②どんな状態／条件で: `.java`ソースコードのメソッド注釈/引数注釈を解析する際に
  ③どうする: 抽出・記録する
  文: `@PathVariable`/`@RequestParam`などの契約依存関係を、`.java`ソースコードのメソッド注釈/引数注釈を解析する際に抽出・記録する
## reason:
  - パス/パラメータ依存を可視化することで、Web APIの契約を把握できる
## description:
  - 用語: パス/パラメータ依存とは、`@PathVariable`/`@RequestParam`などの契約依存を指す
  - 範囲: JavaParserを使用して`@GetMapping`の`value`/`path`や引数注釈名を抽出する
  - 例: `@GetMapping("/users/{id}") public User getUser(@PathVariable Long id)`の場合、パス/パラメータ依存が記録される


## specification_note: 
  - 解析タイプ:  静的解析         
  - 検索対象:  `.java`（メソッド注釈/引数注釈）                   
  - 解析ツール:  JavaParser                      
  - 解析内容:  `@GetMapping`の`value`/`path`や引数注釈名を抽出
