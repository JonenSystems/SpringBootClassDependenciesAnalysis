title: コントローラ定義の解析
## demand:
  ①何が（何を）: Spring MVCでHTTPリクエストを受け取るクラスの依存関係を
  ②どんな状態／条件で: `.java`ソースコードの@RestController注釈を解析する際に
  ③どうする: 抽出・記録する
  文: Spring MVCでHTTPリクエストを受け取るクラスの依存関係を、`.java`ソースコードの@RestController注釈を解析する際に抽出・記録する
## reason:
  - コントローラ定義を可視化することで、Web層の依存関係を把握できる
## description:
  - 用語: コントローラ定義とは、Spring MVCでHTTPリクエストを受け取るクラスを指す
  - 範囲: JavaParserを使用して`@RestController`を検出し、Web依存として分類する
  - 例: `@RestController public class UserController`の場合、UserControllerがWeb層として識別される


## specification_note: 
  - 解析タイプ:  静的解析         
  - 検索対象:  `.java`（@RestController注釈）     
  - 解析ツール:  JavaParser                         
  - 解析内容:  `@RestController`を検出 → Web依存として分類
